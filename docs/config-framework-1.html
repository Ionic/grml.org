1. Introduction

With the config-framework it is possible to save the configuration of a running grml system and restore it again. save-config is a script to create the configuration file(s) and store it on a specific place. restore-config is a script to restore the configuration via using the stored configuration-file. Several bootoptions provide additional features. If you don't know how to use it or where to start take a look at the available options and read through the section Usage Scenarios.

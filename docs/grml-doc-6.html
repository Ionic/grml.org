6.2 Stay up to date

grml is Debian unstable with a few addons. So you can easily stay up to date via debian's packagemanagement. Just run 'apt-get update ; apt-get upgrade' to update packages. In package grml-etc there is a file namend /etc/apt/sources.lists.grml which is always up to date. If you encounter any problems with your /etc/apt/sources.list take a look at the one provided by grml.

6.3 Switch network profile

There exist several possibilities and packages to solve the problem: 

divine:         Automatic IP configuration detection for laptops
guessnet:       Guess which LAN a network device is connected to
ifscheme:       scheme control for network interfaces
ifupdown:       High level tools to configure network interfaces
intuitively:    Automatic IP configuration detection for laptops
laptop-netconf: network detection and configuration program for laptops
laptop-net:     Automatically adapt laptop ethernet
mobilemesh:     Mobile adhoc networking
switchconf:     switch between system configuration sets
whereami:       Automatically reconfigure your (laptop) system for a new location
